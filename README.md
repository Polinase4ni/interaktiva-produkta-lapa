# NPD_I — Mini interneta veikals (interaktīva produkta lapa)

## Projekta apraksts
Šī ir mini “interneta veikala” lapa, kur lietotājs izvēlas preci, daudzumu un piegādi.
JavaScript aprēķina kopējo cenu un parāda rezultātu bez lapas pārlādes.

## Ko lapa prot (funkcionalitāte)
- Preces izvēle (select)
- Daudzuma ievade (input)
- Piegādes veida izvēle (select)
- Promo atlaide (-10%) ar checkbox
- Validācija: lauki nevar būt tukši, daudzumam jābūt veselam skaitlim > 0
- Rezultāta izvadīšana DOM
- Dinamiska stila maiņa, ja kopējā summa ir lielāka par 50 €

## Kā lietot
1) Izvēlies preci
2) Ievadi daudzumu
3) Izvēlies piegādi
4) Ja vajag, atzīmē promo (-10%)
5) Nospied “Aprēķināt cenu”
6) Rezultāts parādīsies uzreiz zem pogas
